## üîç Key Technical Decisions

1. **JWT Authentication**: Chosen for stateless, scalable authentication
2. **React Context**: Used for state management (lightweight, no external library)
3. **Axios Interceptors**: Centralized API calls with automatic token injection
4. **MySQL**: Relational database for data integrity and complex relationships
5. **Vite**: Modern, fast build tool for optimal development experience
6. **Custom CSS**: Full control over design system vs framework constraints

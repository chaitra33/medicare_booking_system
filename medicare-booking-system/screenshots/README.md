## How to Take Screenshots

### Windows

- Press `Windows + Shift + S` to use Snipping Tool
- Or use `PrtScn` key and paste into Paint

### Mac

- Press `Cmd + Shift + 4` for selection screenshot
- Press `Cmd + Shift + 3` for full screen

### Browser Extensions

- **Awesome Screenshot** (Chrome/Firefox)
- **Fireshot** (Chrome)
- Full page screenshots without scrolling

## Tips for Good Screenshots

- Use full browser width
- Clear browser cache for clean state
- Use sample data that looks realistic
- Crop out browser tabs if needed
- Save as PNG for best quality
- Name files exactly as shown above
